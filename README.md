# Flint Water Crisis: Environmental & Community Profile Analysis

Analysis of environmental and demographic data surrounding the Flint, MI water crisis, 
built in R using American Community Survey (ACS) data.

## What this project does
- Pulls 2021 5-year ACS estimates for Flint and surrounding areas
- Profiles demographic and socioeconomic indicators by census tract
- Visualizes disparities in affected communities

## Tools used
R, R Markdown, tidycensus, ggplot2
